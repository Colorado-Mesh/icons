# Icons

Icon assets for Linux, macOS, and Windows along with a colorized version.


# Usage

Create a new logo with:

```sh
./new_icon.sh [SOURCE_FILE] [LOGO_NAME] [SET AS DEFAULT "true"|"false"]
```

If `SET AS DEFAULT` flag is `true`, icon will also be copied to the `default` directory.
